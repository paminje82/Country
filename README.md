# 🌍 지구촌 국기 탐험대 (Country Flag Explorer)

초등학생들이 즐겁게 세계 여러 나라의 국기를 익힐 수 있도록 제작된 인터랙티브 국기 맞추기 게임입니다. 

## 🎯 게임 주요 특징

### 🗺️ 대륙별 탐험 모드
- 아시아, 유럽, 아메리카, 아프리카, 오세아니아 중 원하는 대륙을 선택하여 집중 학습할 수 있습니다.
- 모든 대륙의 국기가 섞여서 나오는 '전체 모드'도 지원합니다.

### ⏱️ 스릴 넘치는 8초 타이머
- 각 문제당 **8초의 제한 시간**이 주어집니다. 
- 시간이 3초 남으면 타이머 바가 분홍색으로 변하며 긴박함을 알려줍니다.
- 시간 초과 시 자동으로 오답 처리되어 순발력이 요구됩니다.

### 🧠 스마트한 퀴즈 엔진
- **중복 방지 시스템**: 최근 10문제 내에 나왔던 국기는 다시 나오지 않도록 설계되어 다양한 국기를 경험할 수 있습니다.
- **콤보 시스템**: 연속으로 정답을 맞히면 🔥 콤보 점수 보너스를 획득합니다.
- **기록 저장**: `LocalStorage`를 활용해 대륙별 최고 점수를 브라우저에 저장하여 도전 욕구를 자극합니다.

### ✨ 프리미엄 디자인
- **반응형 UI**: 태블릿과 PC 환경 모두에서 쾌적하게 즐길 수 있는 직관적인 디자인을 적용했습니다.
- **부드러운 애니메이션**: 화면 전환 및 버튼 호버 시 고급스러운 인터랙션을 제공합니다.

## 🛠️ 사용 기술 및 데이터
- **언어**: HTML5, Vanilla CSS, Modern JavaScript (ES6+).
- **데이터**: [FlagCDN](https://flagcdn.com/)의 고해상도 국기 이미지 API를 실시간으로 활용합니다.

## 🚀 실행 방법
이 저장소를 클론하거나 다운로드한 후, 브라우저에서 `index.html` 파일을 열기만 하면 바로 실행됩니다.
또는, 로컬 서버(예: `http-server`)를 통해 실행하는 것을 추천합니다.
